## ğŸ“ˆ Analytics Calculations

### Average Sleep Duration
```javascript
totalMinutes = sum(all sessions.total_duration_minutes)
avgDuration = totalMinutes / numberOfSessions

Example:
Session 1: 480 min (8h)
Session 2: 420 min (7h)
Session 3: 450 min (7.5h)
Average = (480 + 420 + 450) / 3 = 450 min = 7.5h
```

### Sleep Efficiency
```javascript
actualSleep = total_duration - sum(interruption_durations)
efficiency = (actualSleep / total_duration) * 100

Example:
Total: 480 min (8h)
Interruptions: 15min + 10min = 25min
Actual sleep: 480 - 25 = 455 min
Efficiency: (455 / 480) * 100 = 94.79%
```

### Average Wake-ups
```javascript
totalWakeUps = sum(all sessions.wake_up_count)
avgWakeUps = totalWakeUps / numberOfSessions

Example:
Session 1: 2 wake-ups
Session 2: 1 wake-up
Session 3: 3 wake-ups
Average = (2 + 1 + 3) / 3 = 2 wake-ups per night
```
